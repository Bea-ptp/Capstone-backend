
🥄 Spice & Spoon — Backend API
Backend service for the Spice & Spoon full‑stack capstone project.
Provides REST API endpoints for recipes, spices, authentication, and favorites.

📌 Project Overview
This backend powers the Spice & Spoon web application.
It handles:

. User authentication (JWT)

. CRUD operations for recipes

. CRUD operations for spices

. Favorites management

. Database communication with PostgreSQL

. The backend is built with Node.js, Express, and PostgreSQL.
